# PokedexApp

## Description

This project is one of my first React apps. It's made using PokeApi `https://pokeapi.co/`.
It uses React and two of it's libraries MUI and React-Router.

## Technologies

- React
  - Material UI
  - React Router

## Installation

Requirements:

- Node.js
- Yarn

To install project dependencies use

```
yarn install
```

To run app use

```
yarn run start
```
